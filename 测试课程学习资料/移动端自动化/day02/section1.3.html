课程大纲

1.app元素定位操作API

1.1定位介绍

元素的基本定位基于当前屏幕范围内展示的可见元素

1.2 Appium常用定位方式

  • id 
方法：find_element_by_id(id_value) # id_value:为元素的id属性值

举例：
driver.find_element_by_id("com.android.settings:id/search").click()
  • class
方法：find_element_by_class_name(class_value) # class_value:为元素的class属性值

举例:
driver.find_element_by_id("com.android.settings:id/search").click()
driver.find_element_by_class_name("android.widget.ImageButton").click()
  • xpath
方法:find_element_by_xpath(xpath_value) # xpath_value:为可以定位到元素的xpath语句

Android端常用属性定位:
1. id ://*[contains(@resource-id,'com.android.settings:id/search')]
2. class ://*[contains(@class,'android.widget.ImageButton')]
3. text ://*[contains(@text,'WLA')]
4. contains(@key,value): value可以是部分值

示例:
# driver.find_element_by_xpath("//*[contains(@resource-id,'com.android.settings:id/search')]").click()
# driver.find_element_by_xpath("//*[contains(@class,'android.widget.ImageButton')]").click()
driver.find_element_by_xpath("//*[contains(@text,'更多')]").click()

2. 定位一组元素

应用场景为元素值重复，无法通过元素属性直接定位到某个元素，只能通过elements方式来选择，返回一个
定位对象的列表.

2.1 通过id定位一组元素

title = driver.find_elements_by_id("com.android.settings:id/title")
# 打印title类型，预期为list
print(type(title))
# 取title返回列表中的第一个定位对象，执行点击操作
title[1].click()

2.2 通过class定位一组元素

title = driver.find_elements_by_class_name("android.widget.TextView")
# 打印title类型，预期为list
print(type(title))
# 取title返回列表中的第3个定位对象,执行点击操作
title[3].click()

2.3 通过Xpath定位一组元素

data = driver.find_elements_by_xpath("//*[contains(@class,'android.widget.TextView')]")
data[4].click()

3.显示等待

3.1 显示等待介绍

在一个超时时间范围内，每隔一段时间去搜索一次元素是否存在，
如果存在返回定位对象，如果不存在直到超时时间到达，报超时异常错误。

3.2 显示等待方法

方法:WebDriverWait(driver, timeout, poll_frequency).until(method)
参数：
1.driver：手机驱动对象
2.timeout：搜索超时时间
3.poll_frequency：每次搜索间隔时间，默认时间为0.5s
4.method：定位方法(匿名函数)


匿名函数:
lambda x: x
等价于python函数：
def test(x):
return x

示例:

try:
    # 查找元素前时间
    print(time.strftime("%H:%M:%S",time.localtime()))
    # 显示等待
    WebDriverWait(driver,15,1).until(lambda x: x.find_element_by_id("com.android.settings:id/search")).click()
except Exception as e:
    # 查找元素后时间
    print(time.strftime("%H:%M:%S",time.localtime()))
